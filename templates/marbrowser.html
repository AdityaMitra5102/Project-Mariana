If you are seeing this page, Mariana Daemon has been Installed. Congratulations on joining the Internet of Freedom.

You need the Mariana browser to use Mariana's Qubit. The daemon installer would try to automatically install it (on Windows systems), in which case there would be a Desktop shortcut for the same. Click the button below to download if it has not been automatically installed, for both Windows and Linux systems.



For advanced users not willing to install the Mariana Browser:

Alternatively you can run any web browser with a proxy server at http://localhost:8000 and open http://mariana.local to use Mariana without downloading the Mariana Browser. For example if you have Google Chrome installed, run the following on command prompt

"C:\Program Files\Google\Chrome\Application\chrome.exe" --incognito --proxy-server=http://localhost:8000 http://local.mariana
