If you are seeing this page, Mariana Daemon has been Installed. Congratulations on joining the Internet of Freedom.

You need the a browser with proxy to use Mariana's Qubit. The daemon installer will try to install Firefox automatically (on Windows systems), creating a Desktop shortcut and a Start Menu entry to Mariana.
If it has not been installed automatically download Firefox or Chromium. (Chrome works as well but I do not recommend.)



For users on Linux systems not supporting System wide proxy:

You can run any web browser with a proxy server at http://localhost:8000 and open http://mariana.local to use Mariana. For example if you have Chromium installed, run the following on the terminal

chromium-browser --proxy-server=http://localhost:8000 http://local.mariana



It is recommended to use incognito or in-private mode.
